## 分享功能实现方式

### 1. 好友/群聊分享
通过 `onShareAppMessage()` 生命周期方法实现：

```javascript
onShareAppMessage() {
  return {
    title: '分享标题',
    path: '/pages/xxx/index',
    imageUrl: '/static/share-image.png'
  }
}
```

### 2. 朋友圈分享
通过 `onShareTimeline()` 生命周期方法实现：

```javascript
onShareTimeline() {
  return {
    title: '分享标题',
    query: '',
    imageUrl: '/static/share-image.png'
  }
}
```

### 3. 海报分享
部分页面（如tutor-detail）支持生成海报图片分享：
- 使用canvas绘制海报
- 包含二维码、家教信息等
- 可保存到相册或直接分享

## 分享功能使用建议

### 1. 用户触发分享
- 页面右上角"..."菜单自动显示"转发"选项
- 部分页面提供专门的分享按钮

### 2. 分享内容优化
- 标题简洁明了，突出核心价值
- 路径包含必要参数（如ID）
- 图片清晰美观，符合品牌调性

### 3. 分享追踪
- 可在分享路径中添加来源参数
- 例如：`/pages/teacher-library/index?from=share`
- 用于统计分享效果
