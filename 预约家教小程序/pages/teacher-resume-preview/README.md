# 教师简历预览页面 - 样式优化说明

## iconfont 图标映射

本页面使用了阿里巴巴 iconfont 图标库，以下是图标 Unicode 编码映射：

### 基础图标
- `&#xe60d;` - 用户头像图标
- `&#xe61f;` - 时钟/待审核图标
- `&#xe623;` - 男性图标
- `&#xe622;` - 女性图标
- `&#xe60a;` - 返回箭头图标

### 功能图标
- `&#xe60b;` - 教育/学士帽图标
- `&#xe60c;` - 教学经历/时钟图标
- `&#xe60e;` - 日历/时间图标
- `&#xe610;` - 位置/地图标记图标
- `&#xe611;` - 空状态/文件夹图标
- `&#xe612;` - 个人优势/奖杯图标
- `&#xe613;` - 标签/勾选图标
- `&#xe614;` - 教学风采/相机图标
- `&#xe615;` - 预览/眼睛图标
- `&#xe616;` - 编辑/铅笔图标
- `&#xe617;` - 提交/对勾图标
- `&#xe618;` - 提示/感叹号图标

## 样式优化要点

1. **渐变背景**：顶部使用三色渐变绿色
2. **毛玻璃效果**：使用 backdrop-filter 实现
3. **卡片阴影**：使用 box-shadow 增加层次感
4. **圆角统一**：使用 12-24rpx 的圆角
5. **动画效果**：添加过渡动画和悬停效果
6. **响应式设计**：适配不同屏幕尺寸

## 使用说明

如需更换图标，请访问 [iconfont.cn](https://www.iconfont.cn/) 选择合适的图标，
获取 Unicode 编码后替换模板中的对应编码即可。
