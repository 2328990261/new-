# 教师注册新增字段说明

## 更新时间
2026-02-07

## 新增字段

### 步骤一新增字段

1. **籍贯** (hometown)
   - 类型：字符串
   - 必填：是
   - 说明：从城市数据库中选择
   - 实现：点击弹出城市选择器

2. **教龄** (teaching_years)
   - 类型：数字
   - 必填：是
   - 范围：0-50年
   - 说明：输入数字，自动限制在0-50之间

3. **出生年份** (birth_year)
   - 类型：数字
   - 必填：是
   - 格式：4位数字（如：1995）
   - 验证：不能早于1900年，不能晚于当前年份

4. **所在区域** (location_province, location_city, location_district, location_address)
   - 类型：字符串
   - 必填：是
   - 功能：
     - 点击定位按钮自动获取位置
     - 支持手动修改城市和区县
     - 需要填写详细地址

## 数据库变更

需要执行SQL文件：`tjiajiao91/database/add_teacher_location_fields.sql`

新增字段：
- `hometown` VARCHAR(100) - 籍贯
- `teaching_years` INT(11) - 教龄
- `birth_year` INT(11) - 出生年份
- `location_province` VARCHAR(50) - 所在省份
- `location_city` VARCHAR(50) - 所在城市
- `location_district` VARCHAR(50) - 所在区县
- `location_address` VARCHAR(200) - 详细地址

## API接口

使用现有的地区接口：
- `GET /api/cities/all` - 获取所有城市
- `GET /api/cities/{city_id}/districts` - 获取指定城市的区县

## 功能特点

1. **籍贯选择**：从完整的城市列表中选择
2. **教龄验证**：自动限制在0-50年范围内
3. **出生年份验证**：确保输入有效的年份
4. **智能定位**：
   - 点击定位按钮自动获取当前位置
   - 自动填充省市区和详细地址
   - 支持手动修改
5. **表单验证**：所有新字段都是必填项，不填写无法进入下一步

## 注意事项

1. 定位功能需要用户授权位置权限
2. 如果定位失败，会提示用户手动选择城市
3. 所有字段都会保存到本地缓存，支持断点续填
4. 提交时会一并提交到后端保存
