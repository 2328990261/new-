# 位置信息功能实现说明

## 功能概述
用户登录成功后，系统会请求位置授权，并将位置信息保存到用户数据中，用于推荐附近的优质家教老师。

## 已完成的工作

### 1. 前端实现
- ✅ 创建了位置授权工具文件 `预约家教小程序/utils/location-auth.js`
  - `requestLocationAuth()` - 请求位置授权并保存
  - `saveLocationToUser()` - 保存位置到用户数据
  - `extractProvince/City/District()` - 从地址提取省市区
  - `showLocationAuthDialog()` - 显示授权说明弹窗

- ✅ 在登录页面集成位置授权
  - 登录成功后自动显示位置授权弹窗
  - 用户允许后调用微信的 `chooseLocation` API
  - 获取位置信息（经纬度、地址、省市区）
  - 调用后端API保存到用户数据
  - 更新本地存储的userInfo

- ✅ 在 `预约家教小程序/utils/api.js` 中添加了 `userApi.saveLocation()` API方法

### 2. 后端实现
- ✅ 创建了数据库迁移SQL `new_system/database/add_location_fields_to_wechat_users.sql`
  - 添加位置相关字段：latitude, longitude, address, province, city, district
  - 添加位置索引以提高查询性能

- ✅ 创建了位置保存API `api/user_location.php`
  - 接收用户的openid和位置信息
  - 验证用户是否存在
  - 更新用户的位置信息到数据库
  - 返回保存结果


方法三：在phpMyAdmin中手动执行SQL：
```sql
-- 为微信用户表添加位置信息字段
ALTER TABLE `fa_wechat_users`
ADD COLUMN `latitude` decimal(10,7) DEFAULT NULL COMMENT '纬度' AFTER `user_id`,
ADD COLUMN `longitude` decimal(10,7) DEFAULT NULL COMMENT '经度' AFTER `latitude`,
ADD COLUMN `address` varchar(500) DEFAULT NULL COMMENT '详细地址' AFTER `longitude`,
ADD COLUMN `province` varchar(50) DEFAULT NULL COMMENT '省份' AFTER `address`,
ADD COLUMN `city` varchar(50) DEFAULT NULL COMMENT '城市' AFTER `province`,
ADD COLUMN `district` varchar(50) DEFAULT NULL COMMENT '区县' AFTER `city`;

-- 添加位置索引以提高查询性能
ALTER TABLE `fa_wechat_users`
ADD INDEX `idx_location` (`latitude`, `longitude`),
ADD INDEX `idx_city` (`city`, `district`);
```

### 2. 测试位置授权流程
1. 在小程序中退出登录
2. 重新登录
3. 登录成功后应该看到位置授权弹窗
4. 点击"允许"后选择位置
5. 检查数据库中用户的位置信息是否已保存

### 3. 验证数据保存
在数据库中查询用户位置信息：
```sql
SELECT openid, nickname, latitude, longitude, address, province, city, district 
FROM fa_wechat_users 
WHERE openid = '你的openid';
```

## 工作流程

1. 用户登录成功
2. 显示位置授权说明弹窗："你的位置信息将用于为你推荐附近的优质家教老师"
3. 用户点击"允许"
4. 调用微信 `chooseLocation` API让用户选择位置
5. 获取位置信息：
   - latitude（纬度）
   - longitude（经度）
   - address（详细地址）
   - 从address中提取province、city、district
6. 调用后端API `/api/user_location.php` 保存位置信息
7. 更新本地存储的userInfo
8. 完成登录流程，跳转到相应页面

## 注意事项

1. 位置授权是可选的，用户拒绝不影响登录流程
2. 位置信息保存失败不影响登录流程
3. 位置信息可以用于：
   - 推荐附近的家教老师
   - 按距离排序教师列表
   - 显示教师与用户的距离
   - 地图功能等
