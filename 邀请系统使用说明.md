# 邀请好友系统使用说明

## 功能概述

邀请好友系统允许用户通过邀请码邀请新用户注册，双方在被邀请人认证成功后各获得￥20优惠券。系统包含小程序端、后端API和管理端三部分。

## 主要功能

### 1. 用户端功能（小程序）

#### 邀请页面 (`pages/invitation/index.vue`)
- 显示用户的邀请码
- 展示邀请统计数据：
  - 已邀请人数
  - 已获得优惠券数量
  - 未认证人数
  - 待获得优惠券数量
- 邀请记录列表
- 邀请排行榜 TOP10
- 分享邀请功能

#### 注册使用邀请码页面 (`pages/register-with-invite/index.vue`)
- 输入邀请码
- 验证邀请码有效性
- 绑定邀请关系

#### 优惠券功能
- 查看我的优惠券
- 领取优惠券到卡包
- 优惠券状态：待领取、已领取、已兑换、已过期

### 2. 后端API接口

#### 用户端接口 (`/api/invitation/`)

1. **获取邀请统计** - `GET /stats`
   - 返回用户邀请数据、优惠券统计、邀请列表、排行榜

2. **检查邀请状态** - `GET /check-status`
   - 检查用户是否已使用过邀请码

3. **使用邀请码注册** - `POST /register`
   - 参数：`invite_code` (邀请码)
   - 创建邀请关系

4. **认证回调** - `POST /verify-callback`
   - 用户完成认证后自动调用
   - 发放双方优惠券（各￥20）
   - 更新排行榜数据

5. **领取优惠券** - `POST /receive-coupon`
   - 参数：`coupon_id` (优惠券ID)
   - 将优惠券领取到卡包

6. **我的优惠券** - `GET /my-coupons`
   - 参数：`status` (可选，筛选状态)
   - 返回用户的优惠券列表

#### 管理端接口 (`/admin/invitation/`)

1. **获取概览** - `GET /overview`
   - 总参与人数
   - 总邀请人数、已认证、待认证
   - 优惠券统计（总数、已领取、已兑换、待兑换）
   - 优惠券总金额、已兑换金额

2. **邀请记录列表** - `GET /list`
   - 参数：`page`, `page_size`, `status`, `keyword`
   - 返回邀请记录列表

3. **优惠券列表** - `GET /coupon-list`
   - 参数：`page`, `page_size`, `status`, `keyword`
   - 返回优惠券列表

4. **人工兑换优惠券** - `POST /redeem-coupon`
   - 参数：`coupon_id`, `note`
   - 管理员手动兑换用户优惠券

5. **批量兑换** - `POST /batch-redeem`
   - 参数：`coupon_ids[]`, `note`
   - 批量兑换多张优惠券

6. **搜索用户优惠券** - `GET /search-user-coupons`
   - 参数：`keyword` (用户昵称或手机号)
   - 返回用户及其优惠券信息

7. **排行榜** - `GET /ranking`
   - 参数：`page`, `page_size`
   - 返回邀请排行榜数据

### 3. 管理端功能

#### 邀请管理页面 (`InvitationManage.vue`)

**概览卡片**
- 共多少人参加
- 领取了多少次
- 已兑换/待兑换优惠券
- 总邀请人数、已认证人数
- 优惠券总金额、已兑换金额

**邀请记录标签页**
- 搜索功能（用户昵称/手机号/邀请码）
- 状态筛选（待认证/已认证）
- 显示邀请人、被邀请人信息
- 奖励发放状态

**优惠券管理标签页**
- 搜索功能（用户昵称/手机号）
- 状态筛选（待领取/已领取/已兑换/已过期）
- 单个兑换功能
- 批量兑换功能
- 搜索用户优惠券功能

**排行榜标签页**
- 显示TOP用户
- 总邀请、已认证、待认证人数
- 已领取、已兑换优惠券数量
- 优惠券总金额

## 数据库表结构

### 1. `fa_user_invitations` - 用户邀请表
- 记录邀请关系
- 邀请人、被邀请人信息
- 邀请码
- 认证状态
- 奖励发放状态
- 关联的优惠券ID

### 2. `fa_user_coupons` - 用户优惠券表
- 优惠券信息
- 用户信息
- 优惠券类型、金额
- 来源（邀请人/被邀请人）
- 状态（待领取/已领取/已兑换/已过期）
- 兑换信息（时间、管理员、备注）

### 3. `fa_invitation_ranking` - 邀请排行榜表
- 用户信息
- 邀请统计数据
- 优惠券统计数据
- 排行分数
- 最后邀请时间

### 4. `fa_coupon_redeem_log` - 优惠券兑换记录表
- 兑换记录
- 用户信息
- 管理员信息
- 兑换时间、备注

### 5. `fa_wechat_mini_program_users` - 用户表（新增字段）
- `invitation_code` - 用户的邀请码（唯一）

## 业务流程

### 邀请流程

1. **用户A生成邀请码**
   - 首次访问邀请页面时自动生成6位唯一邀请码
   - 邀请码保存在用户表中

2. **用户A分享邀请码**
   - 通过微信分享
   - 或复制邀请码发送给好友

3. **用户B使用邀请码注册**
   - 用户B输入邀请码
   - 系统验证邀请码有效性
   - 创建邀请记录（状态：待认证）

4. **用户B完成认证**
   - 用户B完成身份/学历/工作认证
   - 系统自动调用认证回调接口
   - 更新邀请记录状态为"已认证"
   - 为用户A和用户B各发放￥20优惠券（状态：待领取）
   - 更新用户A的排行榜数据

5. **领取优惠券**
   - 用户A和用户B在小程序中领取优惠券
   - 优惠券状态变为"已领取"

6. **兑换优惠券**
   - 管理员在管理端搜索用户
   - 查看用户的优惠券
   - 人工兑换优惠券（状态变为"已兑换"）
   - 记录兑换日志

### 优惠券状态流转

```
待领取(0) -> 已领取(1) -> 已兑换(2)
                    |
                    v
                已过期(3)
```

## 安装部署

### 1. 数据库初始化

执行SQL文件：
```bash
mysql -u用户名 -p密码 数据库名 < update_invitation_system_complete.sql
```

### 2. 后端部署

将以下文件部署到后端：
- `app/controller/api/Invitation.php` - 用户端API
- `app/controller/admin/InvitationManage.php` - 管理端API

### 3. 前端部署

#### 小程序端
- `pages/invitation/index.vue` - 邀请页面
- `pages/register-with-invite/index.vue` - 使用邀请码注册页面

#### 管理端
- `views/admin/InvitationManage.vue` - 邀请管理页面

### 4. 配置路由

在管理端路由配置中添加：
```javascript
{
  path: '/admin/invitation-manage',
  name: 'InvitationManage',
  component: () => import('@/views/admin/InvitationManage.vue'),
  meta: { title: '邀请管理' }
}
```

## 注意事项

1. **邀请码唯一性**
   - 系统自动生成6位大写字母+数字的邀请码
   - 确保邀请码在用户表中唯一

2. **认证回调**
   - 用户完成认证后需要调用 `/api/invitation/verify-callback` 接口
   - 建议在认证成功的业务逻辑中自动调用

3. **优惠券过期**
   - 优惠券默认30天有效期
   - 系统在领取时检查是否过期

4. **权限控制**
   - 管理端接口需要管理员登录权限
   - 用户端接口需要用户登录（token验证）

5. **数据统计**
   - 排行榜数据在每次邀请认证成功后自动更新
   - 管理端概览数据实时计算

## 扩展功能建议

1. **优惠券自动兑换**
   - 集成支付系统
   - 用户下单时自动使用优惠券

2. **邀请奖励等级**
   - 根据邀请人数设置不同奖励等级
   - 邀请越多，奖励越高

3. **活动时间限制**
   - 设置活动开始和结束时间
   - 活动期间的邀请才有效

4. **邀请海报**
   - 生成带邀请码的海报
   - 用户可以保存分享

5. **推送通知**
   - 被邀请人注册时通知邀请人
   - 认证成功时通知双方
   - 优惠券到账提醒
