# 部署前准备清单

## 服务器要求

### 最低配置
- **CPU：** 2核心
- **内存：** 2GB
- **硬盘：** 20GB
- **带宽：** 3Mbps

### 推荐配置
- **CPU：** 4核心
- **内存：** 4GB
- **硬盘：** 40GB
- **带宽：** 5Mbps

## 域名准备

需要准备3个域名（或子域名）：

1. **用户端域名：** www.yourdomain.com
2. **管理端域名：** admin.yourdomain.com
3. **API域名：** api.yourdomain.com

## 前端构建准备

### 1. 修改API地址配置

**用户端配置：**
编辑 `new_system/frontend/user/src/api/config.js`：
```javascript
const request = axios.create({
  baseURL: 'https://api.yourdomain.com',  // 改为实际后端域名
  timeout: 10000,
  withCredentials: true
})
```

**管理端配置：**
编辑 `new_system/frontend/admin/src/api/config.js`：
```javascript
const request = axios.create({
  baseURL: 'https://api.yourdomain.com',  // 改为实际后端域名
  timeout: 10000,
  withCredentials: true
})
```

### 2. 构建前端项目

**构建用户端：**
```bash
cd new_system/frontend/user
npm install
npm run build
```

**构建管理端：**
```bash
cd new_system/frontend/admin
npm install
npm run build
```
