# 腾讯地图API配置说明

## 为什么需要配置？

教师注册页面的"点击定位"功能需要将经纬度转换为省市区地址，这需要调用腾讯地图的逆地理编码API。

## 不配置会怎样？

- ✅ 定位功能仍然可以使用（用户可以选择位置）
- ❌ 但无法自动解析为省市区
- 👉 用户需要手动在picker中选择省市区

## 如何申请腾讯地图API密钥？

### 1. 注册腾讯位置服务账号
访问：https://lbs.qq.com/
点击右上角"控制台"，使用QQ或微信登录

### 2. 创建应用
1. 进入控制台后，点击"应用管理" → "我的应用"
2. 点击"创建应用"
3. 填写应用名称（如：家教小程序）
4. 选择应用类型：Web端

### 3. 添加Key
1. 在应用列表中，点击"添加Key"
2. Key名称：随意填写（如：逆地理编码）
3. Key类型：选择"WebServiceAPI"
4. 启用产品：勾选"逆地理编码"
5. 点击"提交"

### 4. 获取Key
创建成功后，会显示一个Key（一串字符），复制这个Key

### 5. 配置到项目
打开 `tjiajiao91/backend/.env` 文件，找到：
```
TENCENT_MAP_KEY = 
```

将Key粘贴到等号后面：
```
TENCENT_MAP_KEY = 你的密钥
```

### 6. 重启后端服务
如果后端正在运行，需要重启才能生效。

## 配置示例

```env
[TENCENT]
# 腾讯地图API密钥
TENCENT_MAP_KEY = XXXXX-XXXXX-XXXXX-XXXXX-XXXXX-XXXXX
```

## 测试是否配置成功

1. 打开小程序
2. 进入教师注册页面
3. 点击"点击定位"
4. 选择一个位置
5. 如果自动填充了省市区，说明配置成功

## 费用说明

腾讯地图API有免费额度：
- 逆地理编码：每天免费调用 **5000次**
- 对于小程序来说，这个额度完全够用
- 超出免费额度后，按次收费（很便宜）

## 常见问题

### Q: 不配置可以吗？
A: 可以，但用户需要手动选择省市区，体验稍差。

### Q: 配置后需要重启吗？
A: 需要重启后端服务（如果正在运行）。

### Q: 密钥会过期吗？
A: 不会，除非你手动删除或禁用。

### Q: 安全吗？
A: 密钥只在后端使用，不会暴露给前端，是安全的。
