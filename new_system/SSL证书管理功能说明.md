# SSL证书管理功能说明

## 功能概述

SSL证书管理功能为家教信息管理系统提供了完整的SSL证书申请、续约和监控解决方案，支持自动化的证书生命周期管理。

## 主要功能

### 1. 证书配置管理
- **域名管理**：支持添加、编辑、删除域名配置
- **证书提供商**：支持Let's Encrypt、阿里云、腾讯云等
- **自动续约**：可配置是否启用自动续约功能
- **启用状态**：可控制证书管理的启用/禁用状态

### 2. 证书申请
- **一键申请**：支持为配置的域名申请SSL证书
- **域名验证**：自动检查域名解析和HTTP访问状态
- **申请状态**：实时显示申请进度和结果

### 3. 证书续约
- **手动续约**：支持手动触发证书续约
- **自动续约**：系统自动检测即将过期的证书并续约
- **批量续约**：支持批量处理多个域名的证书续约

### 4. 状态监控
- **证书状态**：实时显示证书的有效性状态
- **过期提醒**：自动计算并显示证书剩余有效期
- **状态检查**：支持手动检查证书状态

### 5. 日志记录
- **操作日志**：记录所有证书申请、续约操作
- **错误日志**：记录操作失败的原因和详细信息
- **执行历史**：查看历史操作记录

## 使用指南

### 1. 添加域名配置

1. 进入管理端 -> 基础配置 -> SSL证书管理
2. 点击"添加域名"按钮
3. 填写域名信息：
   - **域名**：如 `www.example.com`
   - **联系邮箱**：用于证书申请通知
   - **证书提供商**：选择Let's Encrypt、阿里云或腾讯云
   - **自动续约**：建议开启
   - **启用状态**：开启后才会自动续约

### 2. 申请SSL证书

1. 在域名列表中点击"申请证书"按钮
2. 系统会自动检查：
   - 域名解析是否正确
   - HTTP访问是否可用
3. 申请成功后，证书状态会更新为"有效"

### 3. 证书续约

#### 手动续约
1. 在域名列表中点击"续约"按钮
2. 系统会检查证书是否即将过期（30天内）
3. 续约成功后，证书有效期会延长90天

#### 自动续约
1. 确保域名的"自动续约"开关已开启
2. 系统会每天检查即将过期的证书
3. 自动为符合条件的证书执行续约

### 4. 状态监控

1. 在域名列表中查看证书状态：
   - **待申请**：尚未申请证书
   - **有效**：证书正常有效
   - **过期**：证书已过期
   - **失败**：申请或续约失败

2. 查看过期时间提醒：
   - **绿色**：还有30天以上
   - **蓝色**：还有7-30天
   - **橙色**：还有1-7天
   - **红色**：已过期

### 5. 批量操作

1. 点击"批量续约"按钮
2. 系统会自动处理所有符合条件的证书：
   - 启用自动续约
   - 证书状态为有效
   - 30天内即将过期
3. 查看批量操作结果

## 定时任务配置

### 自动续约脚本

系统提供了自动续约脚本 `ssl_auto_renew.php`，建议配置定时任务：

```bash
# 每天凌晨2点执行自动续约
0 2 * * * /usr/bin/php /path/to/ssl_auto_renew.php
```

### 脚本功能
- 自动查找需要续约的证书
- 检查域名解析和HTTP访问
- 执行证书续约操作
- 记录操作日志
- 发送通知（可选）

## 证书提供商配置

### Let's Encrypt
- **免费证书**：90天有效期
- **自动续约**：支持自动续约
- **域名验证**：HTTP-01验证方式
- **限制**：每周最多20个证书

### 阿里云SSL
- **付费证书**：1年有效期
- **自动续约**：需要API密钥
- **域名验证**：DNS验证或文件验证
- **优势**：稳定性好，支持通配符

### 腾讯云SSL
- **付费证书**：1年有效期
- **自动续约**：需要API密钥
- **域名验证**：DNS验证或文件验证
- **优势**：价格优惠，支持多域名

## 安全注意事项

### 1. 证书存储
- 证书文件存储在安全位置
- 设置适当的文件权限
- 定期备份证书文件

### 2. 私钥保护
- 私钥文件权限设置为600

- 定期轮换私钥

### 3. 访问控制
- 限制SSL管理功能的访问权限
- 记录所有证书操作日志
- 监控异常操作

### 4. 域名验证
- 确保域名解析正确
- 验证HTTP访问可用
- 检查防火墙设置

## 故障排除

### 常见问题

#### 1. 证书申请失败
**可能原因：**
- 域名解析不正确
- HTTP访问被阻止
- 证书提供商API限制

**解决方案：**
- 检查域名DNS设置
- 确认80端口可访问
- 检查API配额限制

#### 2. 自动续约失败
**可能原因：**
- 定时任务未配置
- 脚本执行权限不足
- 网络连接问题

**解决方案：**
- 配置crontab定时任务
- 检查脚本执行权限
- 测试网络连接

#### 3. 证书状态检查失败
**可能原因：**
- HTTPS访问被阻止
- 证书文件损坏
- 服务器配置问题

**解决方案：**
- 检查443端口访问
- 验证证书文件完整性
- 检查Web服务器配置
