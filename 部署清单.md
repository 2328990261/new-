## 部署步骤

### 步骤1：上传后端文件

使用FTP或宝塔面板上传：

1. **Agreement.php**
   - 上传到：`/www/wwwroot/yourdomain.com/app/controller/api/`
   - 确保文件权限：644

2. **api.php**
   - 上传到：`/www/wwwroot/yourdomain.com/route/`
   - 确保文件权限：644

### 步骤2：清除后端缓存

在宝塔终端或SSH中执行：

```bash
cd /www/wwwroot/yourdomain.com
php think clear
```

或者在宝塔面板中：
1. 网站 → 设置 → PHP
2. 点击"重载配置"

### 步骤3：上传前端文件

将 `new_system/frontend/user/dist/` 目录下的所有文件上传到：
```
/www/wwwroot/yourdomain.com/public/
```

**注意：**
- 覆盖现有文件
- 保持目录结构
- assets 目录下的所有文件都要上传

### 步骤5：清除浏览器缓存

**方法1：强制刷新**

1. 打开浏览器设置
2. 清除浏览历史记录
3. 选择"缓存的图片和文件"

## 常见问题

### Q1: 上传后还是404错误？

**检查：**
1. 文件是否上传到正确位置
2. 后端缓存是否清除
3. 浏览器缓存是否清除
4. PHP-FPM是否重启

### Q2: 协议内容不显示？

**检查：**
1. 协议接口是否返回200
2. 浏览器控制台是否有错误
3. 前端文件是否正确上传

### Q3: 支付功能不工作？

**检查：**
1. 支付接口是否可访问
2. 请求路径是否正确（不应该有 /api/api/）
3. 后端日志是否有错误
