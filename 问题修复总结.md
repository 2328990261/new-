**关键修改**:
```javascript
// 构建完整的头像URL
let fullAvatarUrl = serverAvatarUrl
if (serverAvatarUrl && !serverAvatarUrl.startsWith('http')) {
    // 如果是相对路径，添加域名前缀
    if (serverAvatarUrl.startsWith('uploads/')) {
        fullAvatarUrl = envConfig.API_BASE_URL + '/' + serverAvatarUrl
    } else {
        fullAvatarUrl = envConfig.API_BASE_URL + '/uploads/' + serverAvatarUrl
    }
}
```

## 技术细节

### 小程序头像上传流程
1. 用户点击头像触发 `onChooseAvatar` 事件
2. 获取微信返回的临时头像路径
3. 通过 `uni.uploadFile` 上传到后端 `/api/avatar/upload` 接口
4. 后端保存文件并返回真实的头像URL
5. 前端构建完整的头像URL并更新本地存储

### 后端头像上传接口
- 支持通过 `openid` 或 `user_id` 上传头像
- 自动创建上传目录 `uploads/avatars/YYYYMM/`
- 文件类型验证：JPG、PNG、GIF、WebP
- 文件大小限制：最大2MB
- 同时更新 `fa_users` 和 `fa_wechat_users` 表

### 环境配置
- 开发环境：`http://localhost:8000`
- 生产环境：`https://t.jiajiao91.com`
- 自动环境识别和URL构建
