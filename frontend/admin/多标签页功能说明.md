# 多标签页功能说明

## 功能概述

实现了类似浏览器的多标签页系统，标签页显示在顶部 header 中，让用户可以在多个页面之间快速切换。

## 主要特性

### 1. 标签页管理
- **位置**：标签页显示在顶部 header 内，菜单按钮（三条线）右侧
- **动态显示**：只显示已访问过的页面，不预加载未访问的页面
- **所有标签可关闭**：列表页和详情页标签都可以关闭（除了固定的标签）
- **标签切换**：点击标签即可切换到对应页面
- **关闭标签**：点击标签上的 × 按钮关闭
- **拖拽排序**：鼠标拖拽标签可以调整顺序（固定的标签除外）
- **固定标签**：右键菜单可以固定/取消固定标签，固定的标签不能关闭和拖拽
- **智能跳转**：关闭最后一个标签时，自动跳转到线索管理页面
- **右键菜单**：
  - 固定/取消固定标签
  - 关闭当前标签
  - 关闭其他标签
  - 关闭右侧标签
  - 关闭所有标签

### 2. 智能缓存
- 使用 Vue 的 `keep-alive` 组件缓存页面状态
- 切换标签时保持页面的滚动位置、表单数据等
- 关闭标签时自动清理缓存，释放内存

### 3. 用户体验优化
- 标签栏支持横向滚动（标签过多时）
- 活跃标签高亮显示（浅蓝色背景）
- 固定标签特殊样式（浅蓝色边框 + 锁图标）
- 关闭按钮悬停效果（红色背景）
- 平滑的过渡动画
- 简洁美观的标签样式
- 拖拽时的视觉反馈
- 右键菜单快捷操作

## 技术实现

### 文件结构

```
new_system/frontend/admin/src/
├── store/modules/
│   └── tabs.js                   # 标签页状态管理
├── layout/
│   └── index.vue                 # 主布局（已集成标签页）
└── views/admin/
    ├── LeadManage.vue            # 线索管理（已添加 name）
    ├── LeadFollowDetail.vue      # 线索详情（已添加 name）
    ├── TutorManage.vue           # 家教管理（已添加 name）
    ├── TeacherManage.vue         # 教师管理（已添加 name）
    └── OrderManage.vue           # 订单管理（已添加 name）
```

### 核心实现

#### 1. Layout 集成
标签页直接集成在 header 中：
- 位置：菜单按钮右侧
- 样式：圆角卡片式设计
- 交互：点击切换，悬停高亮

#### 2. tabs.js (Pinia Store)
标签页状态管理，提供：
- `tabs`: 标签列表（初始为空，访问时动态添加）
- `activeTab`: 当前活跃标签
- `cachedViews`: 缓存的组件列表
- `addTab()`: 添加标签
- `removeTab()`: 移除标签
- `setActiveTab()`: 设置活跃标签
- `getPageTitle()`: 获取页面标题（简短版本）
- `isDetailPage()`: 判断是否为详情页

### 组件命名

为了让 keep-alive 正确缓存组件，每个页面组件都需要定义 `name` 属性：

```vue
<script>
export default {
  name: 'LeadManage'  // 组件名称
}
</script>

<script setup>
// 组件逻辑
</script>
```

已添加 name 的组件：
- LeadManage
- LeadFollowDetail
- TutorManage
- TeacherManage
- OrderManage

## 使用方式

### 打开详情页
在列表页点击"查看详情"按钮，会自动在新标签中打开详情页：

```javascript
const handleView = (row) => {
  router.push(`/leads/${row.id}`)  // 自动创建新标签
}
```

### 切换标签
直接点击标签栏中的标签即可切换。

### 关闭标签
点击标签上的 × 按钮关闭标签。所有标签都可以关闭（除了固定的标签）。

关闭后会自动切换到前一个标签。如果关闭了最后一个标签，会自动跳转到线索管理页面。

### 拖拽排序
鼠标按住标签拖拽可以调整标签顺序。固定的标签不能拖拽。

### 固定标签
右键点击标签，选择"固定标签"。固定的标签：
- 显示锁图标
- 不能关闭
- 不能拖拽
- 始终显示在最前面

### 右键菜单
右键点击标签，可以：
- **固定/取消固定**：固定常用标签
- **关闭**：关闭当前标签（固定的标签不显示此选项）
- **关闭其他**：关闭除当前标签和固定标签外的所有标签
- **关闭右侧**：关闭当前标签右侧的所有标签（保留固定标签）
- **关闭所有**：关闭所有标签（保留固定标签）

## 样式说明

- 高度：32px

- 超出显示省略号
- 简短命名：

  - 详情页：线索 #123、教师 #456 等

## 注意事项

1. **组件命名**：新增页面组件时，必须添加 `name` 属性才能被正确缓存
2. **路由配置**：详情页路由需要使用动态参数，如 `/leads/:id`
3. **内存管理**：关闭标签时会自动清理缓存，避免内存泄漏
4. **动态加载**：标签页只在访问时添加，不预加载未访问的页面
5. **标题简洁**：使用简短的标题，避免标签过宽
6. **固定保护**：固定的标签不能关闭，确保重要页面始终可访问
7. **智能跳转**：关闭所有标签后，自动跳转到线索管理页面
