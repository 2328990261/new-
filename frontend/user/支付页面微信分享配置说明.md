# 支付页面微信分享配置说明

## 已完成的工作

### 1. 支付页面（Payment.vue）微信分享配置

已为支付页面添加微信分享功能，配置如下：

**分享标题**：信息费支付

**分享描述**：链接已通过安全认证，请放心支付。

**分享链接**：当前页面URL

**分享图片**：网站logo（/logo.png）

## 实现方式

### 代码修改

在 `Payment.vue` 的 `onMounted` 生命周期中添加了微信分享配置：

```javascript
import { setWechatShare } from '@/utils/wechatShare'

onMounted(async () => {
  await loadAgreement()
  
  // 配置微信分享
  setWechatShare({
    title: '信息费支付',
    desc: '链接已通过安全认证，请放心支付。',
    link: window.location.href,
    imgUrl: window.location.origin + '/logo.png'
  })
})
```

### 工作原理

1. **Meta标签设置**
   - 自动设置页面的 meta 标签
   - 包括 Open Graph、微信专用标签、Schema.org 等
   - 适用于所有分享场景

2. **微信JS-SDK集成**
   - 在微信浏览器中自动检测
   - 如果后端配置了微信JS-SDK，会自动初始化
   - 支持分享给好友和分享到朋友圈

3. **多平台兼容**
   - 支持微信、QQ、微博等社交平台
   - 非微信环境下也能正常显示分享信息
   - 降级处理确保兼容性

## 后端配置（可选）

如果需要使用微信JS-SDK的高级功能，需要在后端配置：

### 1. 获取微信公众号配置

- AppID
- AppSecret
- 已认证的公众号

### 2. 配置接口

在 `new_system/backend/app/controller/api/Wechat.php` 中实现：

```php
public function getShareConfig()
{
    $url = $this->request->param('url');
    
    // 获取微信JS-SDK配置
    $config = [
        'enabled' => true,
        'appId' => config('wechat.app_id'),
        'timestamp' => time(),
        'nonceStr' => $this->generateNonceStr(),
        'signature' => $this->generateSignature($url),
        'jsApiList' => [
            'updateAppMessageShareData',
            'updateTimelineShareData'
        ]
    ];
    
    return json(['success' => true, 'data' => $config]);
}
```

### 3. 配置文件

在 `.env` 中添加：

```
WECHAT_APP_ID=your_app_id
WECHAT_APP_SECRET=your_app_secret
```

## 注意事项

### 1. 分享图片

- 建议尺寸：300x300 像素或更大
- 格式：JPG、PNG
- 大小：不超过 32KB（微信限制）
- 路径：确保图片可以公网访问

### 2. 域名配置

如果使用微信JS-SDK，需要：
- 在微信公众平台配置 JS接口安全域名
- 域名必须备案
- 必须使用 HTTPS

### 3. 分享内容规范

- 标题不超过 30 个字符
- 描述不超过 100 个字符
- 不包含诱导分享内容
- 不包含违规关键词

## 常见问题

### Q1: 分享时显示的不是配置的内容？

**A**: 可能原因：
- Meta标签设置不正确
- 微信缓存了旧的分享信息
- 图片URL无法访问

**解决方法**：
- 检查页面源代码中的 meta 标签
- 清除微信缓存（重新关注公众号）
- 确保图片URL可以公网访问

### Q2: 微信JS-SDK配置失败？

**A**: 可能原因：
- 域名未在微信公众平台配置
- signature 签名错误
- timestamp 或 nonceStr 不正确

**解决方法**：
- 检查微信公众平台的域名配置
- 使用微信提供的签名校验工具
- 确保服务器时间正确

### Q3: 非微信环境下能分享吗？

**A**: 可以。系统会：
- 自动设置 meta 标签
- 支持其他社交平台的分享
- 提供复制链接功能
