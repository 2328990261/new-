# 系统部署指南

## 一、管理后台前端打包

### 1. 进入管理后台前端目录
```bash
cd tjiajiao91-main/new_system/frontend/admin
```

### 2. 安装依赖（如果还没安装）
```bash
npm install
```

### 3. 打包生产版本
```bash
npm run build
```

打包完成后，生成的文件在 `dist` 目录中。

## 三、后端部署

### 1. 上传后端文件
将以下目录上传到服务器：
```
tjiajiao91-main/new_system/backend/
```

### 2. 配置数据库
编辑 `.env` 文件，配置数据库连接信息：
```env
APP_DEBUG = false

[DATABASE]
TYPE = mysql
HOSTNAME = 你的数据库地址
DATABASE = 数据库名
USERNAME = 数据库用户名
PASSWORD = 数据库密码
HOSTPORT = 3306
CHARSET = utf8mb4
```

### 3. 设置目录权限
```bash
chmod -R 755 /path/to/backend
chmod -R 777 /path/to/backend/runtime
```

### 4. 配置Nginx（如果使用Nginx）
```nginx
server {
    listen 80;
    server_name your-domain.com;
    root /path/to/backend/public;
    index index.php;

    location / {
        if (!-e $request_filename) {
            rewrite ^(.*)$ /index.php?s=$1 last;
            break;
        }
    }

    location ~ \.php$ {
        fastcgi_pass 127.0.0.1:9000;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        include fastcgi_params;
    }
}
```

## 四、部署前端到服务器

### 方案1：直接上传dist目录
1. 将 `frontend/admin/dist` 上传到服务器的 `/var/www/admin` 目录
2. 将 `frontend/user/dist` 上传到服务器的 `/var/www/user` 目录

### 方案2：使用Nginx配置
```nginx
# 管理后台
server {
    listen 80;
    server_name admin.your-domain.com;
    root /var/www/admin;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}

# 用户端
server {
    listen 80;
    server_name www.your-domain.com;
    root /var/www/user;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

## 七、常见问题

### 1. 前端打包失败
- 检查Node.js版本（建议使用v16或更高）
- 删除 `node_modules` 和 `package-lock.json`，重新安装依赖

### 2. 后端500错误
- 检查 `runtime` 目录权限
- 检查数据库连接配置
- 查看 `runtime/log` 目录中的错误日志

### 3. 跨域问题
- 确保后端已配置CORS中间件
- 检查前端API地址配置是否正确

## 八、快速部署脚本

创建一个 `deploy.sh` 脚本：

```bash
#!/bin/bash

echo "开始部署..."

# 1. 打包管理后台
echo "打包管理后台..."
cd tjiajiao91-main/new_system/frontend/admin
npm run build

# 2. 打包用户端
echo "打包用户端..."
cd ../user
npm run build

# 3. 上传到服务器（需要配置SSH）
echo "上传文件到服务器..."
# scp -r ../admin/dist/* user@server:/var/www/admin/
# scp -r ../user/dist/* user@server:/var/www/user/
# scp -r ../../backend/* user@server:/var/www/backend/

echo "部署完成！"
```

使用方法：
```bash
chmod +x deploy.sh
./deploy.sh
```
