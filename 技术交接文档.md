# 91家教管理系统 - 技术交接文档

## 仓库地址

**Gitee**: https://gitee.com/yuchendeloy/tjiajiao91

## 项目结构

本项目分为两个分支：

- **main 分支**：主系统（后端 + 管理前端 + 用户前端）
- **miniprogram 分支**：微信小程序（uni-app 项目）


## 三、主系统快速开始

### 1. 后端配置

```bash
cd backend

# 安装依赖
composer install

# 复制环境配置
cp .example.env .env

# 编辑 .env 配置数据库信息
# DB_HOST = 127.0.0.1
# DB_NAME = myjiajiao
# DB_USER = root
# DB_PASSWORD = your_password
```

### 2. 数据库初始化

```bash
# 导入数据库
mysql -u root -p myjiajiao < database/create_complete_booking_system.sql
```

### 3. 前端配置

**管理后台**：
```bash
cd frontend/admin
npm install
npm run dev
# 访问 http://localhost:3002
```

**用户前端**：
```bash
cd frontend/user
npm install
npm run dev
# 访问 http://localhost:3001
```

## 四、小程序快速开始

### 1. 切换到小程序分支

```bash
git checkout miniprogram
```

### 2. 使用 HBuilderX 打开

1. 下载 HBuilderX：https://www.dcloud.io/hbuilderx.html
2. 打开 HBuilderX
3. 文件 → 导入 → 从本地目录导入
4. 选择项目目录

### 3. 配置 AppID

编辑 `manifest.json`：
```json
{
  "mp-weixin": {
    "appid": "你的微信小程序AppID"
  }
}
```

### 4. 配置 API 地址

编辑 `config/env.js`：
```javascript
const baseUrl = 'http://localhost:8080'  // 或你的后端地址
```

### 5. 运行

运行 → 运行到小程序模拟器 → 微信开发者工具

## 七、常见问题

### 1. 如何查看所有分支？

```bash
git branch -a
```

### 2. 如何在两个分支之间切换？

```bash
# 切换到主系统
git checkout main

# 切换到小程序
git checkout miniprogram
```

### 3. 如何同时开发两个分支？

**方法 1**：克隆两次到不同目录
```bash
# 主系统
git clone -b main https://gitee.com/yuchendeloy/tjiajiao91.git main-system

# 小程序
git clone -b miniprogram https://gitee.com/yuchendeloy/tjiajiao91.git miniprogram
```

**方法 2**：使用 git worktree
```bash
# 克隆主仓库
git clone https://gitee.com/yuchendeloy/tjiajiao91.git

# 创建 worktree
cd tjiajiao91
git worktree add ../tjiajiao91-miniprogram miniprogram
```

### 4. 后端启动失败？

检查：
- PHP 版本是否 >= 7.4
- MySQL 是否启动
- .env 配置是否正确
- composer 依赖是否安装

### 5. 前端启动失败？

检查：
- Node.js 版本是否 >= 16
- npm install 是否成功
- 端口是否被占用

### 6. 小程序无法运行？

检查：
- HBuilderX 是否安装
- 微信开发者工具是否安装
- AppID 是否配置
- API 地址是否正确
