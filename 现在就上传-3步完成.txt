第 1 步：添加 SSH 公钥到 Gitee（2分钟）
========================================

1. 打开文件：SSH公钥-复制此内容到Gitee.txt
   
2. 复制里面的全部内容（从 ssh-rsa 开始）

3. 访问：https://gitee.com/profile/sshkeys
   
4. 点击"添加公钥"
   - 标题：随便填（如：My Computer）
   - 公钥：粘贴刚才复制的内容
   - 点击"确定"

✅ 完成！


第 2 步：测试 SSH 连接（30秒）
========================================

打开命令行（CMD 或 PowerShell），执行：

ssh -T git@gitee.com

如果提示：
  "Are you sure you want to continue connecting?"
  
输入：yes 然后回车

如果看到：
  "Hi xxx! You've successfully authenticated..."
  
✅ 说明配置成功！


第 3 步：上传代码（1分钟）
========================================

在命令行中执行：

cd e:\123.207.52.46_8088\new_system

git add . 2>nul

git commit -m "feat: 91家教管理系统初始版本"

git push -u origin main

✅ 上传完成！

========================================
如果遇到问题
========================================

问题 1: 第 1 步添加公钥时提示"公钥格式错误"
解决：确保复制了完整的公钥内容，包括开头的 ssh-rsa 和结尾的 gitee_key

问题 2: 第 2 步测试连接失败
解决：等待 1-2 分钟后重试，Gitee 需要时间同步公钥

问题 3: 第 3 步 git add 报错
解决：在 Kiro 外部的命令行中执行，或者重启电脑后再试

问题 4: git push 提示权限错误
解决：返回第 1 步，重新检查公钥是否正确添加
