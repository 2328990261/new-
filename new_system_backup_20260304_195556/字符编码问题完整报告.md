## 🔧 技术细节

### 字符编码问题类型

#### 类型1：中文字符损坏
```
找家教，就这�?    → 找家教，就这么简单
管理后�?           → 管理后台  
轮播�?             → 轮播器
```

#### 类型2：HTML标签不完整
```vue
<!-- 错误 -->
<h1>找家教，就这�?<span class="highlight">简�?/span></h1>

<!-- 正确 -->
<h1>找家教，就这<span class="highlight">么简单</span></h1>
```

#### 类型3：引号缺失
```javascript
// 错误
meta: { title: '退款申? }

// 正确  
meta: { title: '退款申请' }
```

#### 类型4：文件被清空
```
文件大小: 0 字节

```
